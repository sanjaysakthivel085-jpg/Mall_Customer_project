# Mall_Customer_Project

## Overview

This project uses K-Means Clustering to segment customers based on their annual income and spending behavior.

## Objective

* Analyze customer behavior
* Identify customer groups
* Generate business insights

## Tools Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Dataset Features

* CustomerID
* Age
* Gender
* AnnualIncome
* SpendingScore
* PurchaseFrequency

## Methodology

1. Data Collection
2. Data Preprocessing
3. Elbow Method
4. K-Means Clustering
5. Customer Segmentation
6. Data Visualization

## Results

The customers were divided into 5 segments based on income and spending patterns.

## Conclusion

Customer segmentation helps businesses understand customer behavior and create targeted marketing strategies.
